# Tests

Tests are run using [tape](https://npmjs.com/tape).

Linting is performed using [standard](https://npmjs.com/standard)

## Running tests

```console
git clone {this repo}
cd {this repo}
npm install
npm test
```

### Only run the tests

```console
npm test
```

### Only run the linter

```console
npm run lint
```

### Run both tests and the linter

```console
npm run test-lint
```
